# Технологический стек

## Язык программирования
*   **TypeScript:** Основной язык программирования, обеспечивающий строгую типизацию и улучшенную поддержку инструментов разработки.

## Фреймворк
*   **NestJS:** Прогрессивный фреймворк Node.js для создания эффективных и масштабируемых серверных приложений.

## База данных
*   **PostgreSQL:** Мощная, объектно-реляционная система баз данных с открытым исходным кодом.

## ORM
*   **Prisma:** ORM нового поколения для Node.js и TypeScript, упрощающий взаимодействие с базой данных.

## Транспорт
*   **gRPC:** Высокопроизводительный фреймворк для удаленного вызова процедур (RPC) с открытым исходным кодом.

## Менеджер пакетов
*   **Yarn:** Быстрый, надежный и безопасный менеджер зависимостей
